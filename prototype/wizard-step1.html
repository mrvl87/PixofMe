STEP 01

PROJECT SETTING

Satu workspace dapat menyimpan banyak project. Data instansi, pekerjaan, lokasi, dan item pekerjaan disimpan di level project agar bisa dipakai ulang pada laporan berikutnya.

Arah UX

  • Workspace menjadi rumah untuk project_1, project_2, dan seterusnya.
  • Project Setting menyimpan atribut utama project.
  • Item RAB otomatis menjadi dropdown di Step 3.
